## 📍 Overview

The Cartoon Character Project is designed as a showcase website where users can explore different cartoon characters in a structured card layout. Each character is presented with an image, name, short description, and a “More About” button for better interaction.
## 🌟Features

- **Card-based layout for displaying cartoon characters.**
- **Images with character names and descriptions.**
- **Navigation bar (Home, Cartoons).**
- **Interactive “More About” buttons.**
- **Clean, simple, and user-friendly design.**
