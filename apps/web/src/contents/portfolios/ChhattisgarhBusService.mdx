## 📍 Overview

The Chhattisgarh Bus Service project is a multi-page web application designed to provide users with a complete and interactive experience for exploring bus transportation within the state. The project begins with an engaging loading screen featuring a video, followed by a visually appealing homepage with a background video that enhances the overall user experience. On the homepage, users can search for their travel route by selecting a starting location such as Durg and choosing their destination. Based on the selected route, the application navigates to another page where users can view available buses along with their timings and basic details. After selecting a specific bus, users are directed to a detailed route page that displays the complete journey path, including stops and timing information. This structured flow allows users to move step-by-step from searching routes to exploring detailed travel information, making the application user-friendly and practical.

## 🌟 Features

- **Loading screen with video animation for better first impression.**
- **Homepage with background video for enhanced visual experience.**
- **User-friendly interface with clean and structured layout.**

- **Navigation to bus listing page after route selection.**
- **Display of multiple buses with timings and details.**
- **Detailed bus route view showing stops and journey path.**
- **Step-by-step navigation across multiple pages.**
- **Organized information flow for better usability.**
- **Interactive UI elements for smooth user experience.**
- **Real-world based transportation system concept.**
- **Responsive design for different screen sizes.**
- **Simple and intuitive user interaction.**
