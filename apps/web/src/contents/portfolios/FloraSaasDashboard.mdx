## 📍 Overview

This project includes multiple screens such as login, signup, and dashboard interface, all designed with a consistent floral theme. The UI focuses on soft color palettes, smooth layouts, and visually appealing components to create a calm and professional user experience. Each screen is uniquely designed while maintaining consistency in typography, spacing, and visual hierarchy. The project demonstrates strong design thinking, creativity, and the ability to create themed interfaces for modern web applications.

## 🌟 Features

- **SaaS dashboard UI with floral theme.**
- **Login and signup page design.**
- **Clean and elegant user interface.**
- **Soft color palette and aesthetic visuals.**
- **Consistent design system across screens.**
- **Focus on typography, spacing, and hierarchy.**
- **Modern and minimal design approach.**
- **User-friendly layout and navigation.**
