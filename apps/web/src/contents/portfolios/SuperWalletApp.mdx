## 📍 Overview

This project includes multiple mobile screens such as wallet overview, balance display, payment options, QR scanning, and transaction history. The design emphasizes simplicity and usability, ensuring that users can easily navigate through different features like sending money, receiving payments, and tracking transactions. Special attention has been given to layout structure, spacing, and component design to create a smooth and intuitive user experience similar to real-world fintech applications.
## 🌟 Features

- **Mobile wallet UI design with multiple screens.**
- **Dashboard showing total balance and quick actions.**
- **Pay and Receive functionality interface.**
- **QR code scanning screen for payments.**
- **Transaction history screen.**
- **Clean and modern mobile UI design.**
- **Consistent color scheme and layout..**
- **Focus on usability and user experience.**
